url 模块

本文总阅读量: 次

url模块用于生成和解析URL。该模块使用前，必须加载。

url.parse()

url.parse() 将一个 URL 字符串转换为URL对象

url.format()

url.format() 方法允许将一个 URL 对象转换为 URL 字符串

url.resolve(from, to)

url.resolve方法用于生成URL。它的第一个参数是基准URL，其余参数依次根据基准URL，生成对应的位置。
