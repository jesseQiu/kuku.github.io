Css 学习札记

本文总阅读量: 次

主要是记录学习极客学员的 CSS3 视频教程笔记，知识点比较零散，主要是为了做个备忘

一、选择器

  • *{} 通配符选中所有 元素
  • h1,h2…{} 元素选择器
  • .className{} 类选择器
  • #idName{} id 选择器，只能有一个，不能结合使用
  • 后代选择器 空格
  • 第一子代选择器 >
  • 相邻兄弟选择器
  • h1+p{} // 如果需要选择紧接在另一个元素后的元素，而且二者有相同的父元素，可以使用相邻兄弟选择器
  • 多类元素选择器
  • 属性选择器
  • [title]{} // 选择元素具有 title 属性的元素
  • [href=”http://jessechiu.com"]{} // 只匹配属性 href 值为 “Jessechiu.com” 的元素
  • [value~=”jesse”]{} // 匹配只要属性有 value 且值中有包含 “jesse”，类似 [value*=”jesse”]

  • li:only-child{} // 选择只有一个 li 子元素的 li 元素

  • 伪类元素选择器(css3 中总共有 17 个)

  • input[name=”jesse”]:hover/focus/active/checked/invalid/valid{} //
  • input[name=”jesse”]:enabled/disabled/required/selection/optional{} //
  • input[name=”jesse”]:read-only/default/indeterminate/in-range{} //
  • 通用兄弟选择器
    div ~ p{} // 同一父元素之中，某个元素之后，所有其它其它元素，例如

二、文本样式

  • text-align: center/left/right;
  • text-wrap: normal;// 正常是换行，当时必须和 width 一起设置使用
  • text-shadow: 1px 1px 1px blue;// 阴影距离 左、顶、清晰度、颜色，可以同时设置多个阴影

三、字体样式

  • font-size: xxpx;
  • font-family: ‘微软’ ‘宋体’;
  • 设置自定义字体

四、a 标签样式

  • a:link 普通的，未被访问的链接
  • a:visisted
  • a:hover
  • a:active 链接被点击时
  • text-decoration:none 用于去掉 a 标签下划线

五、list 样式

  • list-style-type: none; // 可以设置列表前面的点样式，none 为无。
  • list-style-image:url();// 可以设置列表项图标

六、表格样式

  • 设置表格边框为折叠边框，这样边框只显示一条线

七、CSS 轮廓 // 主要用来突出元素作用

  • outline-style:dotted;
  • outline-color:blue;
  • outline-width:2px; // 设置轮廓粗细

八、CSS 定位

  • position: static/relative/absolute/fixed
  • float: left/right/none/inherite
  • clear: left/right/both; // 去掉浮动，受影响的元素

九、盒子模型

  • 宽、高：是指 内容 的宽、高
  • 内边距
  • border-style: none/silid/…; 有十种边框样式
  • border-radius: 1px;
  • border-shadow:1px 1px 1px blue; // 类似 text-shadow
  • border-image: url();
  • 外边距
  • 默认为透明区域
  • 通过 margin 属性设置
  • 两个外边距合并，取大的值

十、对齐操作

  • margin: 0px auto; // 水平居中，前提条件需要设置 width
  • position
  • float

十一、常用操作

  • line-height: normal/200%/50%; 设置行高
  • max-width/height: 1px; // 设置最大值，在换行是会自动拆分单词
  • cursor: point; // 设置鼠标样式
  • display: inline/block; // 设置显示方式，可以把 li 设置显示为水平方式，用于导航栏
  • visibility: visible/hidden; // 设置是否可见
  • opacity: 0-1; // 设置图片透明度，0：完全透明，1：不透明

十二、CSS 动画

  • transform: translate(100px,100px); // 表示向 X 坐标、向 Y 坐标，移动

  • transform: rotate(45deg); // 旋转角度

  • transform: scale(1,2); // 放大，第一个是 宽度，第二是高度 倍数

  • transform: skew(45deg,30deg); // 倾斜角度，向 x 轴，向 y 轴

  • transform: rotateX/Y(45deg); // 3D 旋转效果

  • 过渡

  • transition: 5s;// 过渡时间
  • 动画
  • animation: anim 5s infinite alternate; // anim 是动画名称，对应下面的 @keyframes 动画帧名，infinite 是无限循环，alternate 可以让动画往返动作
  • 多
  • columns: 200px 3; // 列宽度、列数
  • column-gap: 10px; // 列间距
  • column-rule：2px outset red; // 规定列之间的宽度、样式和颜色
    // ref：http://www.w3school.com.cn/cssref/pr_columns.asp

十三、CSS3 盒子模型

  • display:inline/inline-block/block

  • display:inline-table // 可以设置 table 以 inline 方式显示

  • display:list-item// 可以设置元素按照 li 方式显示，这样就有 li 相对应的样式属性设置

  • overflow:hidden/scroll/auto // auto 会自动根据文字内容，在水平和垂直方向添加滚动条

  • overfolow-x/y:hidden/scroll/auto // 也可以单独在 x或者y 轴上设置

  • box-shadow: 1px 1px 1px red;// 设置盒子阴影，使用方法和 text-shadow 相同。

  • 指定盒子大小计算方法

  • box-sizing: border-box;// border 和 padding 包含在盒子大小中
  • box-sizing: content-box;// border 和 padding 不包含在盒子大小中，显示会大

十四、CSS3 背景相关属性

  • background-clip: border-box;// 背景色充满 border 和 padding

  • background-clip: padding-box;// 背景色充满 padding

  • background-clip: content-box;// 背景色 只填充内容部分

  • background-origin:border/padding/content-box; // 代表背景从那个区域块开始绘制

  • 同一元素绘制多张图片

  • 使用图片边框，背景图片会根据里面内容自动撑开

  • -webkit-border-image:url() 10 20 10 30;
    // ref: http://www.w3school.com.cn/cssref/pr_border-image.asp
